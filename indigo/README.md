Indigo (Violet 2.0.0)
=====================

Indigo is the codename for the next major version of Violet.
We don't expect this version to be realised any time soon, and that's why it's developed through a different directory.

## Versions

| Indigo Version | Violet Version |
| -------------- | -------------- |
| 1.x.x          | 2.x.x          |

## Installation

```bash
npm install lex-indigo --global
indigo install
```
