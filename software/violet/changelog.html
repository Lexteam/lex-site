Violet Changelog

1.2.1

Released on 18th January 2017.

This version has a blog post.

Changes:

  • Removed bullet points from nav-bar
  • Improved spacing on nav-bar
  • Underline on hover on nav-bar items

1.2.0

Released on 4th May 2016.

This version has a blog post.

Changes:

  • Complete colour classes, with both British and American spelling
  • Small tweaks to blog: heading and subheading now centred.
  • Now using Neat and Bourbon
  • Completely new nav bar design
  • Much more customisation ability.
  • Proper mobile support for the nav bar
  • Images in the blog-content are now mobile friendly
  • Less margin for panels on mobile devices

Additions:

  • Add grid system
  • Add portals - made to contain links new areas of the site (with mobile support)
  • Add British spelling of centre
  • Add support for printing pages (doesn't display the nav bar)

1.1.1

Released on 3 March 2016.

Changes:

  • Allow for more customisation of the nav bar
  • Removed some none-Violet related classes (this is potentially breaking, and if you are using violet, our apologies)
  • More variables to customise Violet with (we are working towards more customisation than just colours)
  • Style blockquotes

Additions:

  • Add colour classes
  • New page related classes (.content)

1.1.0

Released on 27 February 2016.

This version has a blog post and was released alongside the violet-cli.

Changes:

  • New colour scheme
  • Fonts changed
    • Base font set to Open Sans
    • Header font set to Raleway
  • Change how pages work

1.0.1

Released on 15 February 2016.

Changes:

  • New colour scheme

1.0.0

Released on 30 December 2015.

Initial release.
