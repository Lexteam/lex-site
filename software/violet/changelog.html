Violet Changelog

1.1.1

Released on 3 March 2016.

Changes:

  • Allow for more customisation of the nav bar
  • Removed some none-Violet related classes (this is potentially breaking, and if you are using violet, our apologies)
  • More variables to customise Violet with (we are working towards more customisation than just colours)
  • New page related classes (.content)
  • Style blockquotes

1.1.0

Released on 27 February 2016.

This version has a blog post and was released alongside the violet-cli.

Changes:

  • New colour scheme
  • Fonts changed
    • Base font set to Open Sans
    • Header font set to Raleway
  • Change how pages work

1.0.1

Released on 15 February 2016.

Changes:

  • New colour scheme

1.0.0

Released on 30 December 2015.

Initial release.
